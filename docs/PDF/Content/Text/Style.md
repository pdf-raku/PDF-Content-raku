### multi method baseline-shift

```raku
multi method baseline-shift(
    "alphabetic"
) returns Numeric
```

compute a vertical offset for a named font alignment mode

This returns a positive or negative y-offset in units of points. The default is `alphabetic`, which is a zero offset.
