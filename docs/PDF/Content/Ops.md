class PDF::Content::Ops
-----------------------

A graphics machine for building and interpeting PDF Content operator streams

-----------

```raku

    :$delta!
```

```raku

```

```raku
method current-point() returns PDF::Content::Ops::Vector
```

### multi method op

```raku

    Pair $_ where { ... }
) returns Mu
```
